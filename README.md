A level building tool for Geometric Wave

## How To Launch Tool

### Prerequisites
- Node.js installed on your system

### Install
- Install dependencies (first time only, or after pulling updates):
   ```
   npm install
   ```

### Steps to Launch
1. Open a terminal/command prompt in the project directory:
   ```
   e:\Unity\Projects\GeometricWave\GeometricWave_LevelBuildingTool
   ```

2. Start the server:
   ```
   npm start
   ```

   The server will start and run on the default port (check [server.js](server/server.js) for port configuration).

### Updating Packages
To update packages to the latest compatible versions:
```
npm update
```

To check for outdated packages:
```
npm outdated
```
